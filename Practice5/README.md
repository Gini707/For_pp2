# Practice 5 – Regex and Receipt Parsing

This practice contains Python programs that use regular expressions.

Files:
- receipt_parser.py – parses receipt data from raw.txt
- raw.txt – raw receipt text
- 1.py – task 1 (a followed by zero or more b)
- 2.py – task 2 (a followed by two or three b)
- 3.py – lowercase words with underscore
- 4.py – uppercase followed by lowercase
- 5.py – string starting with a and ending with b
- 6.py – replace spaces, commas and dots with colon
- 7.py – convert snake_case to camelCase
- 8.py – split string at uppercase letters
- 9.py – insert spaces between capital words
- 10.py – convert camelCase to snake_case

Regular expressions are used to match, search, and transform text patterns.
